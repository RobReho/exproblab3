Description:
The node subscribes to the topic /aruco_marker_publisher/ID When an ID is published on that topic, if it is a new ID, it will retrieve the corresponding hint calling the service server /oracle_hint. Then hint will then be inspected: only the properly formed hints will go to the state machine. The good hints are published on the topic /good_hint and will be retrieved there by the state machine
